# Temperature Conversion Calculator

This is a simple calculator to convert from celsius and fahrenheit with two primary functions:
1. Celsius -> Fahrenheit
2. Fahrenheit -> Celsius

It does not accept invalid inputs.

# Features
- Separate callable methods for temperature conversion.
- Input validation ("Invalid Input!")

# How to Run
Java JDK 17 or higher must be installed.

1. Open a terminal and ensure you are in the correct folder: TemperatureConverter-Java
2. Compile the program: javac Main.java
3. Run the program: java Main

You should see the following prompt:

Temperature Converter
Option 1: Fahrenheit -> Celsius
Option 2: Celsius -> Fahrenheit
Please enter 1 or 2 for the conversion type:  

You may follow whichever option you like!
