# Cooking Converter

This is a simple Python script that helps you convert between different cooking units like cups, tablespoons, grams, and more. It uses tkinter.

## How to Run

Run this in your terminal:

    python converter.py

## Example Conversions

- 1 Cup (c) to Milliliters (mL):
  - Enter `1` in Value
  - Select `Cups (c)` for From
  - Select `Milliliters (mL)` for To
  - Click Convert
  - Result: `236.5900 Milliliters (mL)`

- 2 Tablespoons (tbsp) to Teaspoons (tsp):
  - Value: `2`
  - From: `Tablespoons (tbsp)`
  - To: `Teaspoons (tsp)`
  - Result: `6.0000 Teaspoons (tsp)`

- 100 Grams (g) to Ounces (oz):
  - Value: `100`
  - From: `Grams (g)`
  - To: `Ounces (oz)`
  - Result: `3.5276 Ounces (oz)`

## Notes
- This only works for basic cooking units.
- If you type something that's not a number, you'll get an error.
