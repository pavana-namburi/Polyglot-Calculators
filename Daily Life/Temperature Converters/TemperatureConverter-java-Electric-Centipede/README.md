# Temperature Converter (Java)

A simple command-line temperature converter that supports conversion between Celsius and Fahrenheit, written in **Java 17**.

## How to Use

1. Run the program.
2. Enter `1` to convert **Celsius to Fahrenheit**  
   or `2` to convert **Fahrenheit to Celsius**.
3. Input the temperature value when prompted.
4. The program will display the converted result.
