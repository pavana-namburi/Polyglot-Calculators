# Temperature Converter with Javascript

This is a conversion tool for temperature. It can convert from Celsius to Fahrenheit or from Fahrenheit to Celcius.

# How it Works

In the terminal type: 

    node TemperatureConverter-Javascript/script.js

It will then ask you which type of Conversion you want to do. 
    - If you want to convert C->F, type: c
    - If you want to convert F->C, type: f

Next, it will ask you to type in the temperature you are converting. It will then return the converted temp.
