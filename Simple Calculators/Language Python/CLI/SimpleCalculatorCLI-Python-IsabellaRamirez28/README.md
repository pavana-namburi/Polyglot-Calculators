# Simple Calculator

This is a simple **Command Line Interface (CLI) Calculator** written in python.

This are the arithmetic operations supported:

- Addition
- Subtraction
- Multiplication
- Division

## Features

- **Input:** Accepts integers and float numbers (Positive and negative)
- Safe division (Prevents division by zero)
- The program **does not accept** inputs different of numbers.
- In the divition operation the first number is divided by the second one

## How to run de the calculator:
### **1. Install python or check that you have python installed: **
You need Python 3.x
check with: python --version

### **2. Navigate to the project folder**
open your terminal and go to the directory where **main.py** is located: Awesome-Calculators\Simple Calculators\Language Python\CLI\SimpleCalculatorCLI-Python-IsabellaRamirez28\main.py

### **3. Run the calculator**
Use the following comands depending on your version:
- python main.py
- python3 main.py

When running the program, you will see: 
=== Simple Calculator CLI ===
1. Addition
2. Subtraction
3. Multiplication
4. Division
5. Exit

Just select the option and follow the next instructions.
