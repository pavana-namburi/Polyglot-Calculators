# Project 
A simple Calculator used for basic arithmetic calculations

## Implementations
- Consists of Basic Maths operators:
- Addition(+)
- Subtraction(-)
- Division( / )
- Multiplication (*)

## Usage
- Performing Addition, Subtraction , Multiplication , Division between two Numbers

## Format
Format must be:
- Selecting the desired operator among (+ - * /)
- Input for Number1 and Number2

## Requirements
- Any C++ compiler works

### Compile the Code
Navigate to the directory containing `main.cpp` and use the C++ compiler (`g++`) to build the executable:
```
g++ main.cpp -o calculator
./calculator
```
