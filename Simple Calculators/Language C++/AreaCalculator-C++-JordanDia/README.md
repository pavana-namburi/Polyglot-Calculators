# AreaCalculator-Cpp

A simple **C++ console program** to calculate the area of basic geometric shapes: Circle, Rectangle, Triangle, and Square.

---

## Features

- Interactive console input
- Supports:
  - Circle (requires radius)
  - Rectangle (requires width and height)
  - Triangle (requires base and height)
  - Square (requires side length)
- Continuously runs until the user chooses to quit
- Clean and readable C++ code

---

## Usage

1. Compile the program:

```bash
g++ main.cpp -o area_calc
```

2. Run the program:

```bash
./area_calc
```

3. Examples:

```bash

1. Circle
2. Rectangle
3. Triangle
4. Square
```
