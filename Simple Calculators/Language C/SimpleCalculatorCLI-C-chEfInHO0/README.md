
# Simple Console Calculator in C

A basic, robust command-line calculator written in C, designed to perform the four fundamental arithmetic operations. This project was created as a contribution to the Awesome-Calculators repository to fulfill Issue #68.

## ‚ú® Features

* **Four Basic Operations:** Supports Addition (`+`), Subtraction (`-`), Multiplication (`*`), and Division (`/`).
* **Zero-Division Checking:** Includes validation to prevent division by zero errors.
* **Robust Input Handling:** Functions for input handling ensure that the program correctly processes only numeric values and valid operators.
* **Clean Code:** Modularized functions to separate input/output logic from arithmetic logic.

Run the compiled executable from your terminal:
